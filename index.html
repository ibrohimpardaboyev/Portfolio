Wiasabi Bank Atms Trancation Analysis

Technologies: Power BI, Microsoft Fabric, Sql

I’ve been working on an ATM Utilization & Transaction Analytics Dashboard, and it’s been a great learning project! This helped me turn raw financial data into clear, actionable insights, making it easier to understand customer behaviors and transaction trends.

Key components included:

  • Transaction trends – When and where people use ATMs the most
  • Demographic insights – Who is making transactions (age, occupation, etc.)
  • Financial activity patterns – Withdrawals, deposits, transfers, and balance inquiries
  • Geospatial analysis – ATM utilization rates by state

Banks and financial institutions can use these insights to optimize ATM locations, improve services, and enhance customer experience. Even though this is a demo project, it demonstrates how data visualization can drive better financial decisions.
